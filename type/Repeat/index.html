Repeat

struct Repeat<T> { ... }

A collection whose elements are all identical Ts.

Inheritance

CollectionType Protocol hierarchy

Associated types

typealias Index = Int

A type that represents a valid position in the collection.

Valid indices consist of the position of every element and a "past the end" position that's not valid for use as a subscript.
